## Project Description
This project simulates a two-road traffic light controller using Arduino Uno. The traffic lights automatically switch between Red, Yellow, and Green signals according to a predefined timing sequence.

## Components Used
- Arduino Uno
- 6 LEDs
- 6 × 220Ω Resistors
- Jumper Wires
- Wokwi Simulator

## Software Used
- Arduino IDE
- Wokwi Simulator

## Project Output

The simulation demonstrates a two-road traffic intersection.

Road A:
- Green → Yellow → Red

Road B:
- Red → Green → Yellow

The sequence repeats automatically.
